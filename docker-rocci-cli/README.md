# rOCCI Client tool

## Introduction
The docker image is Ubuntu 14.04 LTS with rOCCI client tool

## Build the docker image

```
docker build -t rocci-cli .
```

Or you can pull from the dockerhub

```
docker pull rocci-cli
```
